# A.S.Launcher

<概要>  
   アーマースタンドを前方(プレイヤーの向き)に射出するデータパックです。  
  「人参付きの棒(カスタムモデルデータ:1)」を使う事で、発射出来ます。  
 
  対象のアイテムは、次のコマンドで入手してください。  
  give @s minecraft:carrot_on_a_stick{CustomModelData:1}  


<注意>  
  他のデータパックと競合する可能性があるので、単体で試す事をお勧めします。
